# MIST
4ES3 L02

Final project of SMRTTECH 4ES3 course.

A PCB project. Has an LED matrix that lights up in patterns based on music input from a microphone. Can make robot faces that makes expressions. Made with KiCad v9.0.

See /CODE to see the programming of the PCB.
